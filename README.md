# # Sistema de Gestión de Calificaciones

Este programa fue desarrollado en C++ para gestionar las calificaciones de estudiantes.

## Funciones del programa

- Registrar estudiantes y calificaciones
- Mostrar promedio general
- Mostrar nota más alta
- Mostrar nota más baja
- Mostrar estudiantes aprobados (>=70)

## Tecnologías utilizadas

- Lenguaje C++
- Arreglos
- Funciones
- Menú interactivo
